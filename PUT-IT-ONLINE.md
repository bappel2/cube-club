# Put Cube Club online (GitHub Pages)

You have a GitHub account already, so this takes about 5 minutes. No coding, no cost, no domain to buy.

## Try it first (no internet needed)
Just double-click **index.html** — it opens in your browser and fully works. You can hand it to your son right now. To use it on a tablet offline, open it once in the browser, then use the browser's "Add to Home Screen" option.

## Get it a web address with GitHub Pages

1. Go to **github.com** and sign in.
2. Click the **+** in the top-right → **New repository**.
3. Name it something like `cube-club`. Set it to **Public**. Click **Create repository**.
4. On the new repo page, click **uploading an existing file** (the link in the middle).
5. Drag **index.html** into the box. (You can drag the other files too, but only index.html is required.) Click **Commit changes**.
6. Click **Settings** (top menu) → **Pages** (left menu).
7. Under "Build and deployment," set **Source** to **Deploy from a branch**, pick branch **main** and folder **/ (root)**, then **Save**.
8. Wait about a minute, refresh the page. GitHub shows a link like:
   **https://YOURNAME.github.io/cube-club/**

That link is the site. Bookmark it on his tablet. Done!

## Add or change videos (grown-up job)
Open **index.html** in any text editor (Notepad is fine). Near the top of the script find the **VIDEOS** list. Each line has a title and a YouTube link. Replace the example links with real ones you've watched and approved, then re-upload index.html to GitHub (repo → index.html → pencil icon → paste → Commit, or drag-upload again).

The kid can only reach the videos you put in that list — there's no open YouTube inside the app.

## Good to know
- His times and cube collection are saved **on his device**, in his browser. They stay between visits. Using a different device or clearing browser data starts fresh.
- Everything is private — no accounts, no personal info collected, no ads.
- Want changes (more solve steps, a real 3D cube, badges, a different look)? Just ask.
