# Cube Club 🧩

A free, kid-friendly Rubik's cube web app for young beginners (ages ~6–9). One `index.html`, no build step, no server, no accounts — works offline and hosts free on GitHub Pages.

## Features
- **⏱️ Speed Timer** — spacebar/hold-to-start timer with scrambles, best & average, and a celebration on a new best.
- **🧠 Learn to Solve** — a 9-step beginner guide with cube diagrams and big, simple move notation.
- **📺 Watch Videos** — a curated, parent-controlled video shelf (no open YouTube).
- **🎁 My Cubes** — a collection binder for the cubes he owns, with favorites.

## Run it
Open `index.html` in any browser. That's it.

See `PUT-IT-ONLINE.md` for a 5-minute GitHub Pages walkthrough.

## Develop
See `CLAUDE.md` for architecture, data model, conventions, and roadmap. Everything is vanilla HTML/CSS/JS in `index.html` — keep it buildless.

## Privacy
No accounts, no analytics, no ads, no personal data collected. Times and collection are saved locally in the browser.
